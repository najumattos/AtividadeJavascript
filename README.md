# 🧹 Atividade de Refatoração - Desenvolvimento de Sistemas II Prof. Lucas

Este repositório contém a resolução de 9 exercícios de refatoração propostos na disciplina de Desenvolvimento de Sistemas II. O objetivo principal foi aplicar boas práticas de programação, transformar códigos legados em estruturas modernas e facilitar o entendimento dos conceitos aplicados.

LUCAS SE VOCE LER ISSO, EU MEREÇO MB 

## 🚀 O que foi explorado
A refatoração não foi apenas visual. Durante o processo, apliquei conceitos fundamentais de JavaScript moderno (ES6+) e arquitetura de código:

- **Modernização de Variáveis**: Substituição de var por let e const para garantir escopo de bloco e imutabilidade onde necessário;

- **Dry (Don't Repeat Yourself)**: Identificação e eliminação de códigos duplicados;

- **Funções Especialistas**: Divisão de funções complexas em funções menores que fazem apenas uma coisa (Single Responsibility Principle);

- **Sintaxe Moderna**: Uso de Arrow Functions, Template Literals e Operadores Ternários para reduzir a verbosidade;

- **Estruturas de Controle**: Substituição de múltiplos if/else por switch case onde a lógica permitia maior clareza;

- **Desacoplamento**: Desaninhamento de funções para melhorar a testabilidade e leitura;

- **Padronização**: Uso de PascalCase em novas adições e componentes específicos.


## 📖 Como estudar por este repositório
O projeto foi estruturado para servir como material de apoio didático:

- **Versão Comentada**: No estado atual do código, você encontrará explicações detalhadas ("tim-tim por tim-tim") sobre o porquê de cada mudança, além de perguntas reflexivas como "O que isso faz?" ou "Existe outra maneira de fazer?".

- **Histórico de Commits**: Para ver o código "limpo" (sem os comentários de estudo), você pode navegar pelo histórico de commits. Isso permite comparar o antes e o depois da refatoração.
